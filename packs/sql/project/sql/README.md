# SQL

## Convenções

- Uma query por arquivo quando possível.
- Nome do arquivo no padrão: `dominio_objetivo.sql`.
- Explicitar colunas.
- Documentar granularidade e chaves de join.
- Evitar `SELECT *` em produção.

## Cabeçalho recomendado

```sql
-- Objetivo:
-- Fonte(s):
-- Destino:
-- Granularidade:
-- Chaves:
-- Filtros obrigatórios:
```
