description: Use esta skill para reduzir consumo de tokens/contexto em tarefas com Copilot e agentes.
---

Estratégia para economizar tokens:

1. Ler primeiro arquivos de orientação: README, AGENTS.md e instruções específicas.
2. Buscar por termos relevantes antes de abrir muitos arquivos.
3. Abrir apenas arquivos diretamente relacionados.
4. Fazer plano curto antes de editar.
5. Evitar reescrever arquivos grandes sem necessidade.
6. Não colar logs completos; resumir e focar no trecho do erro.
7. Usar instruções por caminho em vez de colocar todas as regras no arquivo global.
8. Usar skills para fluxos ocasionais.
9. Usar agentes especialistas para tarefas grandes, mantendo o agente principal focado.
