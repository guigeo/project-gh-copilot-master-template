Estrutura recomendada de README:

1. Nome do projeto
2. Objetivo
3. Arquitetura resumida
4. Requisitos
5. Instalação
6. Configuração
7. Execução
8. Testes
9. Estrutura de pastas
10. Troubleshooting
11. Próximos passos

Regras:

- Use português brasileiro.
- Inclua comandos copiáveis.
- Não exponha segredos.
- Mantenha README curto e mova detalhes para `docs/`.
